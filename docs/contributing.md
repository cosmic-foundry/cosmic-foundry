# Contributing

The developer workflow (environment setup, branch and PR discipline,
commit style, ADR process) lives in
[`CONTRIBUTING.md`](https://github.com/cosmic-foundry/cosmic-foundry/blob/main/CONTRIBUTING.md)
in the repository root. It is authored as GitHub-rendered markdown and is
not mirrored into these docs.

See also
[`AI.md`](https://github.com/cosmic-foundry/cosmic-foundry/blob/main/AI.md)
for the agent-specific guidelines that apply to AI contributors, and
[`ARCHITECTURE.md`](https://github.com/cosmic-foundry/cosmic-foundry/blob/main/ARCHITECTURE.md)
for the architectural decisions those guidelines flow from.
